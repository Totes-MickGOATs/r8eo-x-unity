# AI Knowledge Base

This directory contains documentation for AI agents working on this project.

## Structure

- `knowledge/architecture/` -- System overview, coding standards, ADRs
- `knowledge/plans/` -- Implementation plans for major features
- `knowledge/research/` -- Research notes, competitive analysis, technology evaluation
- `knowledge/status/` -- Project status, phase tracking, test checklists

## How to Use

### For AI Agents
- Read `knowledge/status/project-status.md` first to understand the current phase
- Check `knowledge/architecture/system-overview.md` for architectural context
- Review `knowledge/architecture/coding-standards.md` before writing code
- Check `knowledge/plans/` for any existing plans related to your task

### For Developers
- Add new plans using the template: `knowledge/plans/TEMPLATE.md`
- Document research using: `knowledge/research/TEMPLATE.md`
- Record architectural decisions using: `knowledge/architecture/adr-TEMPLATE.md`
- Keep `knowledge/status/project-status.md` updated as phases complete

### Templates
All `TEMPLATE.md` files are fill-in-the-blank starting points. Copy them, rename, and fill in your specifics.
