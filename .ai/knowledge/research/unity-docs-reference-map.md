## Notable Unity 6.x Changes

- **Scriptable Audio Pipeline** — new audio architecture for custom DSP processing
- **Unity AI** (Sentis, Generators, Assistant) — ML inference and AI-assisted development
- **Building Blocks** — LiveOps and multiplayer starter kits
- **Multiplayer Widgets** — pre-built UI for multiplayer features
- **visionOS** — Apple Vision Pro platform support
- **Embedded Systems** — Linux/QNX embedded platform support
- **Version numbering** — rebranded from "2023.x" to "Unity 6" (6000.x)
