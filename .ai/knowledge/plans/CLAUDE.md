# knowledge/plans/

Implementation plans for major features and milestones.

## Files

| File | Role |
|------|------|
| `TEMPLATE.md` | Template for new implementation plans |
| `feature-brief-TEMPLATE.md` | Structured template for milestone planning before coding |

Add new plans by copying `TEMPLATE.md` (or `feature-brief-TEMPLATE.md` for milestones) and filling in specifics.
