# .claude/

Claude Code configuration: hooks, commands, settings, and statusline.

## Files

| File | Role |
|------|------|
| `settings.json` | Permissions, hooks, worktree config, statusline |
| `guard-master-commit.sh` | PreToolUse hook: blocks git commit on main branch |
| `statusline.sh` | Status bar: model, cost, tokens, context %, daily analytics |

## Subdirectories

| Dir | Contents |
|-----|----------|
| `hooks/` | Event hooks: lint-on-save, session lifecycle, quality gates |
| `commands/` | Slash commands organized by category (ci/, dev/) |
| `analytics/` | Daily cost/token tracking (auto-generated by statusline) |
