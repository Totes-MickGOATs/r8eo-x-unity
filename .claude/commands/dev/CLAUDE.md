# commands/dev/

Development slash commands for Claude Code.

## Commands

| Command | Description |
|---------|-------------|
| `add-system` | Create a new game system with manifest, scripts, tests |
| `bulletproof` | 6-phase quality checklist for code changes |
| `check-performance` | Performance audit for game systems |
| `e2e-test` | End-to-end test planning and execution |
| `init-project` | Interactive first-time project setup walkthrough |
| `next-task` | Find the next task to work on |
| `review-code` | Code review checklist |
| `run-tests` | Run tests for specific files or systems |
| `test-session` | Interactive testing session |
| `validate-docs` | Check CLAUDE.md freshness across the project |
