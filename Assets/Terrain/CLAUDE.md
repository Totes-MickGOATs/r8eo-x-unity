# Assets/Terrain/

Terrain data, heightmaps, and terrain textures organized by track.

## Subdirectories

| Dir | Purpose |
|-----|---------|
| `Outpost/` | Outpost test track — 100m×100m, 10m height |

## Outpost Track

| File | Role |
|------|------|
| `Outpost/Data/heightmap.raw` | 16-bit RAW heightmap (2049×2049) from displacement.png |
| `Outpost/Data/OutpostTerrainData.asset` | Unity TerrainData (generated by editor script) |
| `Outpost/Textures/displacement.png` | Original 16-bit displacement for reference |
| `Outpost/Textures/edge-mask.png` | Grayscale blend mask (from TGA alpha) for dirt layer blending |
| `Outpost/Textures/normal-map.png` | Macro normal map (RGB from TGA) |
| `Outpost/Textures/specular-map.png` | Macro specular map (from TGA alpha) |

## Build Instructions

1. Open Unity Editor
2. Menu → RC Buggy → Build Outpost Track
3. Terrain is created centered at origin, 100m×100m, 10m tall
