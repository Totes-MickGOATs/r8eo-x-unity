# unity-damage-progression/

Vehicle damage modeling, component wear simulation, and career progression systems for RC racing.

## Files

| File | Contents |
|------|----------|
| `SKILL.md` | Damage zones, visual damage (shader/decal/LOD swap), physics modifiers, repair system, tire wear, battery/motor simulation, career mode tiers, vehicle upgrades, currency economy, unlock system, progressive tuning disclosure, implementation priorities |

## Relevant Skills

| Skill | Relationship |
|-------|-------------|
| `unity-scriptable-objects` | Template pattern used for upgrades, tire compounds, and unlock conditions |
| `unity-save-load` | Serialization of VehicleState, career progress, and currency to JSON |
| `unity-physics-3d` | Physics foundations for collision detection and force-based damage |
| `unity-physics-tuning` | Vehicle Rigidbody config that damage modifiers apply to |
