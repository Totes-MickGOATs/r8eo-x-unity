# unity-weather-conditions/

Dynamic weather, track conditions, and environmental effects for RC racing simulation.

## Files

| File | Contents |
|------|----------|
| `SKILL.md` | Weather state machine, global wetness, rain/dust/mud systems, wind physics, time of day, temperature, track evolution, wet shaders, performance tips, priority table |

## Relevant Skills

| Skill | Relationship |
|-------|-------------|
| `unity-physics-3d` | General physics foundations this skill's grip/drag calculations build on |
| `unity-physics-tuning` | PhysX configuration for the vehicle that weather modifiers apply to |
| `unity-particles-vfx` | Particle system fundamentals for rain, dust, and mud effects |
| `unity-3d-materials` | Shader and material basics for wet surface rendering |
