# unity-genre-racing

Comprehensive skill for building racing games in Unity — vehicle physics, track design, race management, AI opponents, multiplayer networking, input handling, and camera systems. Focuses on RC racing but patterns apply to any vehicle-based game.

## Files

| File | Contents |
|------|----------|
| `SKILL.md` | Full reference: vehicle physics architecture, Pacejka tire model, track design, checkpoint systems, AI racing, camera systems, input handling, multiplayer, RC-specific considerations |
