# engine/godot/commands/editor/

Godot editor slash commands copied to `.claude/commands/editor/` by setup-engine.sh.

## Commands

| Command | Description |
|---------|-------------|
| `editor-health` | Check Godot editor health and connectivity |
| `inspect-node` | Inspect a node in the scene tree |
| `project-info` | Display project settings and configuration |
| `scene-build` | Build or modify a scene programmatically |
| `scene-tree` | Display the current scene tree hierarchy |
| `screenshot` | Capture editor viewport screenshot |
